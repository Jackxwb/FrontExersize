这个文档主要讲述的是斜着的条纹背景

下面代码这样写，看效果是怎样的。 

    
      background: linear-gradient(45deg, #fb3 50%, #58a 0);
      background-size: 30px 30px;

实际上我们没有将最小的重复单位给绘出来,实际上最小的重复单位是这样的

咱们将代码修改一下： 

    
      background: linear-gradient(45deg, #fb3 25%, #58a 0, #58a 50%, #fb3 0, #fb3 75%, #58a 0);
      background-size: 30px 30px;

但是这个宽度明显不对的。这个size 应该是 2 * 根号(30/2) ，最小单位的长度和宽度是30，根据勾股定理来算，这个宽度和长度不是30，是前面算出来的值
